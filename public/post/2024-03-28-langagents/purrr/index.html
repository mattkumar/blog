Langchain

Enter Langchain. Specifically Langchain Agents.

I’m not going to go into an exposition on them (read the docs üìóüìòüìô) but I want to show how we can leverage them to do things for us.

At a high level, I’ll create a “tool” üî® to do a task. I’ll make this tool available to the LLM via Langchain. When the agent is invoked, it will reason whether to use my tool (or not ü§î) based on my prompt üóØ

My First Tool

Let’s call him Smith üë§

Smith is a Python üêç function that will:

  • Access a public API that suggests activities to someone who is bored üì°
  • Return a random response üîô
  • Parse the response to a string üë∑
def Smith():
    res = requests.get('https://www.boredapi.com/api/activity?participants=1')
    return res.json()['activity'] + ' Key:' + res.json()['key']

Here’s an example of a return:

We get a single activity recommendation, followed by a reference üîë to verify the message came from the API. See here: link

So our tool Smith() does what we want. The task at hand now is to make Smith() available to the LLM. And that’s where Langchain comes in.

LLMs and Smith

In this section I’ll show very briefly how to include Smith() inside of a Langchain agent call:

from langchain_openai import ChatOpenAI
from langchain.tools import BaseTool
from langchain.agents import initialize_agent
import requests

llm = ChatOpenAI()

class Smith(BaseTool):
    name = "suggestions for boredom"
    description = "useful for when you need to give suggestions for boredom."

    def _run(self, query: str) -> str:
        res = requests.get('https://www.boredapi.com/api/activity?participants=1')
        return res.json()['activity'] + ' Key:' + res.json()['key']

agent = initialize_agent(llm = llm, 
                         tools = [Smith()], 
                         agent = "zero-shot-react-description", 
                         verbose = True)

That’s about it. Lets try running it with the following prompt and view the output:

agent.run("I am bored, suggest me a few activities. Provide me a key for each activity so I can reference it later.")

üí• Pretty neat, right? This output log shows the reasoning of the agent through Action, Observation, Thoughts steps until it arrives at a final answer. The point here is that it used Smith() to reply because it was determined the tool was the right fit üíØ
